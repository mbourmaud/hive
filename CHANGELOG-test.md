# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.2.0] - 2026-02-12

### Added
- WebUI dashboard with real-time drone monitoring
- Export drone logs to CSV format
- Dark mode theme support

### Fixed
- Fixed memory leak in drone connection pool
- Corrected timezone handling in activity logs

## [1.1.0] - 2026-01-15

### Added
- Multi-drone status overview page
- Configurable refresh intervals for dashboard
- Search functionality for drone logs

### Changed
- Improved UI responsiveness on mobile devices
- Updated drone connection retry logic

## [1.0.0] - 2026-01-01

### Added
- Initial release of WebUI for Hive
- Basic drone monitoring dashboard
- Activity log viewer
- Real-time status updates
