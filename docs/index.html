Hive

Drone orchestration for Claude Code. Launch autonomous agents on PRD files while you keep working.

curl -fsSL https://raw.githubusercontent.com/mbourmaud/hive/main/install.sh | bash

Get Started in 5 Steps

1

Install

One command installs the CLI and Claude Code skills.

curl -fsSL https://raw.githubusercontent.com/mbourmaud/hive/main/install.sh | bash
2

Init

Creates the .hive/ folder structure in your project.

hive init
3

PRD

Generate a PRD with stories using Claude Code.

/hive:prd
4

Start

Launch a drone on your PRD. It works autonomously.

hive start --prd .hive/prds/my-feature.json
5

Monitor

Watch your drones work. Check status and logs anytime.

hive status && hive logs my-feature
