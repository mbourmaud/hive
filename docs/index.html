Hive

You're The Queen. They're The Drones.
Put Them To Work.

1
Install
One command installs the CLI and all Claude Code skills.
curl -fsSL https://raw.githubusercontent.com/mbourmaud/hive/main/install.sh | bash
2
Initialize
Creates the .hive/ folder in your project to store PRDs and drone state.
hive init
3
Create a PRD
Within Claude Code
Generate a PRD with stories from your feature description.
/hive:prd
4
Launch a Drone
Within Claude Code
Start an autonomous Claude agent on your PRD. It creates a git worktree and works independently.
/hive:start
5
Monitor
Within Claude Code
Check on your drones anytime. View status and logs in real-time.
/hive:status
6
Setup Statusline
Within Claude Code
Track all drone progress directly in your statusline.
/hive:statusline
Optional — see drone progress without leaving your editor
