How it Works

Get your hive running in three simple steps

01

Install Hive CLI

One-line installation via curl. Works on macOS and Linux.

curl -fsSL https://...install.sh | bash
02

Initialize Your Project

Run hive init to set up Docker compose, git worktrees, and configuration.

03

Start Your Swarm

Launch your Queen and Workers with hive start N. Connect and orchestrate!
